Best Fit

 

 

Best Fit tool helps us to find the theoretical model (from TSAP server) that best fits a given Spectral Energy Distribution (from SSAP server/s).

The best model is the model that achieves the global minimum of the Chi-square function. To solve this minimization problem this tool uses a modified version of the iterative Levenberg-Marquardt method.

TSAP server and initial parameters are selected by the user.

Chi-squared values and parameters are shown in the fitting window for every step during the minimization process.

 

A maximum number of iterations is defined in case of no convergence.

If the best fit model is found, the user can choose to load and view it in the display window.

 

  

 

Note: TSAP servers with continuous parameters are currently not supported by Best Fit.
